# Car Rental Management System (C)

A terminal-based system to manage customers, vehicles, and car reservations with persistent data saving.

## Features
- Add, modify, delete customers and cars
- Make or cancel reservations
- Store data in .txt files

## Tech Stack
- C
- File I/O (text files)

## How to Run
Compile with any C compiler (e.g., GCC):
```bash
gcc main.c -o car_rental
./car_rental
